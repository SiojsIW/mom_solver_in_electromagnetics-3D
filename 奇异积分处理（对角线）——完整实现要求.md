# 第 6 步：奇异积分处理（对角线）——完整实现要求

---

## 一、格林函数分解（提取两项）

### 分解公式

$$G(R) = \frac{e^{-jkR}}{4\pi R} = \underbrace{\frac{1}{4\pi R}}_{\text{强奇异}} \underbrace{- \frac{k^2 R}{8\pi}}_{\text{一阶导不连续}} + \underbrace{G_{\text{smooth2}}(R)}_{\text{二阶光滑}}$$

### 光滑核定义

$$G_{\text{smooth2}}(R) = \frac{e^{-jkR} - 1 + \dfrac{k^2 R^2}{2}}{4\pi R}$$

**Taylor 验证**：$R \to 0$ 时，$G_{\text{smooth2}} \to -\dfrac{jk}{4\pi} + \dfrac{jk^3 R^2}{24\pi} + O(R^3)$，是 $C^2$ 光滑函数。

### 接口

在 `GREEN_FUNCTIONS.F90` 中：

```fortran
COMPLEX FUNCTION GREEN_FUNC_SMOOTH2(R, RP, K) RESULT(G)
```

### 变量定义

| 变量    | 类型     | 含义                        |
| :------ | :------- | :-------------------------- |
| `DIST`  | 实数标量 | 两点欧氏距离                |
| `PHASE` | 复数标量 | $e^{-jk \cdot \text{DIST}}$ |

### 算法步骤

1. 计算 `DIST` = 欧氏距离。
2. 若 `DIST < 1.0E-10`：返回 $-jk/(4\pi)$。
3. 否则：计算 `PHASE` = $e^{-jk \cdot \text{DIST}}$，返回 `(PHASE - 1 + 0.5*K²*DIST²) / (4π*DIST)`。

### 验证标准

- `DIST = 0`：实部 ≈ 0，虚部 = $-K/(4\pi)$。
- `DIST = 0.01, K = 2π`：与 Taylor 展开对比，误差 $< 10^{-8}$。

## 二、解析标量势（单三角形）

### 目标

计算：
$$I_0(\mathbf{r}) = \int_T \frac{1}{|\mathbf{r} - \mathbf{r}'|} dS'$$

### 接口

在 `SINGULAR_INTEGRAL.F90` 中：

```fortran
SUBROUTINE ANALYTIC_SCALAR_POT_1_OVER_R(R_PT, V1, V2, V3, RESULT)
```

### 变量定义

| 变量               | 类型        | 含义                                       |
| :----------------- | :---------- | :----------------------------------------- |
| `NORMAL`           | 3维实数向量 | 外法向（右手定则归一化）                   |
| `RESULT`           | 实数标量    | 累积结果，初始化为0                        |
| `I`                | 整数        | 边索引，1到3                               |
| `V_START`, `V_END` | 3维实数向量 | 当前边起点和终点                           |
| `EDGE_VEC`         | 3维实数向量 | 边向量 = 终点 - 起点                       |
| `L_I`              | 实数标量    | 边长                                       |
| `T_HAT`            | 3维实数向量 | 边单位方向向量                             |
| `M_HAT`            | 3维实数向量 | 边内法向（平面内，指向内部）               |
| `H_I`              | 实数标量    | 有符号距离 = (场点-起点)·M_HAT             |
| `S_START`          | 实数标量    | 场点相对于起点的有符号投影                 |
| `S_END`            | 实数标量    | 场点相对于终点的有符号投影 = S_START - L_I |
| `R_START`, `R_END` | 实数标量    | 场点到起点/终点的距离                      |
| `V_OPP`            | 3维实数向量 | 对顶点（当前边不连接的顶点）               |
| `DOT_CHECK`        | 实数标量    | 验证内法向是否指向内部                     |

### 算法步骤

1. 计算外法向 `NORMAL` = $(\mathbf{v}_2-\mathbf{v}_1) \times (\mathbf{v}_3-\mathbf{v}_1)$，归一化。
2. 对三条边循环（I = 1, 2, 3）：
   - 确定 `V_START` = 顶点 I，`V_END` = 顶点 I+1（模3）。
   - 计算 `EDGE_VEC`、`L_I`、`T_HAT`。
   - 计算 `M_HAT` = `NORMAL` × `T_HAT`。
   - 确定 `V_OPP`（三个顶点中不是起点也不是终点的那个）。
   - 验证：`DOT_CHECK` = (`V_OPP` - `V_START`) · `M_HAT`。若 `DOT_CHECK < 0`，则 `M_HAT` = -`M_HAT`。
   - 计算 `H_I` = (`R_PT` - `V_START`) · `M_HAT`。
   - 计算 `S_START` = (`R_PT` - `V_START`) · `T_HAT`。
   - 计算 `S_END` = `S_START` - `L_I`。
   - 计算 `R_START` = |`R_PT` - `V_START`|，`R_END` = |`R_PT` - `V_END`|。
   - 若 `R_START + S_START > 1.0E-12`：
     - `CONTRIB` = `H_I` × ln( (`R_END` + `S_END`) / (`R_START` + `S_START`) )
   - 否则（数值不稳定）：
     - `CONTRIB` = `H_I` × ln( (`R_START` - `S_START`) / (`R_END` - `S_END`) )
   - `RESULT` = `RESULT` + `CONTRIB`。
3. 返回 `RESULT`。

### 验证标准

- 等边三角形（边长1）+ 重心场点：`RESULT` > 0，量级约 0.4~0.5。
- 与 50点高斯积分对比：误差 $< 10^{-10}$。
- 场点在内部：所有边的 `H_I` 同号，`RESULT` 为正。

## 三、解析线性势（单三角形）

### 目标

计算三个线性势：
$$I_i(\mathbf{r}) = \int_T \frac{\xi_i(\mathbf{r}')}{|\mathbf{r} - \mathbf{r}'|} dS', \quad i = 1, 2, 3$$

其中 $\xi_i$ 是面积坐标（在顶点 $i$ 处为1，其余为0）。

### 接口

```fortran
SUBROUTINE ANALYTIC_LINEAR_POT_1_OVER_R(R_PT, V1, V2, V3, RESULTS_3)
```

### 变量定义

复用要求一的所有变量，新增：

| 变量        | 类型        | 含义                      |
| :---------- | :---------- | :------------------------ |
| `RESULTS_3` | 3维实数向量 | 三个线性势结果，初始化为0 |
| `I0_TEMP`   | 实数标量    | 临时标量势（用于验证）    |

### 算法步骤

公式来源：Graglia 1993, Eq. (25)–(27)。

对每条边 $i$（从 $\mathbf{v}_i$ 到 $\mathbf{v}_{i+1}$），计算对三个顶点 $j=1,2,3$ 的贡献。

每条边的贡献形式与标量势类似，但多了：
- $h_i^2$ 的加权项
- $R_i$ 的线性项（与端点距离成正比）
- 面积坐标的端点取值（0或1）

**具体公式结构**（对每条边）：

对顶点 $j$（$j$ 是边的起点、终点、或对顶点），该边对 $I_j$ 的贡献为：

$$\text{CONTRIB}_{i,j} = \frac{h_i^2}{2} \cdot \xi_j^{\text{(edge)}} \cdot \ln\left(\frac{R_i^{\text{end}} + s_i^{\text{end}}}{R_i^{\text{start}} + s_i^{\text{start}}}\right) + \frac{1}{2} \cdot \xi_j^{\text{(edge)}} \cdot \left(s_i^{\text{end}} R_i^{\text{end}} - s_i^{\text{start}} R_i^{\text{start}}\right) + \text{(与顶点位置相关的线性项)}$$

其中 $\xi_j^{\text{(edge)}}$ 是面积坐标在边端点处的取值（起点处 $\xi_{\text{start}}=1$，其余为0）。

**简化实现策略**：
1. 先完成要求一（标量势），确保边循环框架正确。
2. 查阅 Graglia 1993 Eq. (25)–(27)，在相同的边循环框架内，为每个顶点 $j$ 累加对应的贡献项。
3. 数值稳定处理与标量势相同。

### 验证标准（最关键）

1. **恒等式**：$I_1 + I_2 + I_3 = I_0$（因为 $\xi_1 + \xi_2 + \xi_3 = 1$）。
2. **对称性**：等边三角形 + 重心场点 $\Rightarrow I_1 = I_2 = I_3 = I_0 / 3$。
3. 若上述两条不满足，说明公式实现有误。

## 四、EFIE 同一三角形自阻抗

### 目标

计算单个三角形 $T$ 对自身的完整 EFIE 阻抗贡献。

### 接口

在 `Z_MATRIX.F90` 中：

```fortran
SUBROUTINE CALC_EFIE_SELF_TRI_PAIR(MESH, TRI_ID, RWG, GDATA, K, ETA0, Z_PAIR)
```

### 变量定义

| 变量                         | 类型        | 含义                                                         |
| :--------------------------- | :---------- | :----------------------------------------------------------- |
| `C`                          | 实数标量    | RWG 在该三角形上的系数                                       |
| `R_OPP`                      | 3维实数向量 | 该三角形上的对顶点坐标                                       |
| `A`                          | 实数标量    | 三角形面积                                                   |
| `JW_MU`                      | 复数标量    | $jk\eta_0$                                                   |
| `J_WE`                       | 复数标量    | $j\eta_0/k$                                                  |
| `P_SING`                     | 复数标量    | P算子 $1/R$ 解析部分                                         |
| `Q_SING`                     | 复数标量    | Q算子 $1/R$ 解析部分                                         |
| `P_SMOOTH2`                  | 复数标量    | P算子 $G_{\text{smooth2}}$ 数值部分                          |
| `Q_SMOOTH2`                  | 复数标量    | Q算子 $G_{\text{smooth2}}$ 数值部分                          |
| `I0_VAL`                     | 实数标量    | 当前场点的标量势                                             |
| `I1_VAL`, `I2_VAL`, `I3_VAL` | 实数标量    | 当前场点的三个线性势                                         |
| `IVEC_VAL`                   | 3维实数向量 | 向量势 = V1·I1 + V2·I2 + V3·I3                               |
| `SUM_I0`                     | 实数标量    | 外层高斯累积：$\sum w_p I_0(\mathbf{r}_p)$                   |
| `SUM_R_I0`                   | 3维实数向量 | 外层高斯累积：$\sum w_p \mathbf{r}_p I_0(\mathbf{r}_p)$      |
| `SUM_IVEC`                   | 3维实数向量 | 外层高斯累积：$\sum w_p \mathbf{I}_{\text{vec}}(\mathbf{r}_p)$ |
| `SUM_R_DOT_IVEC`             | 实数标量    | 外层高斯累积：$\sum w_p [\mathbf{r}_p \cdot \mathbf{I}_{\text{vec}}(\mathbf{r}_p)]$ |
| `J0`                         | 实数标量    | `SUM_I0` × `A`                                               |
| `J_R_I0`                     | 3维实数向量 | `SUM_R_I0` × `A`                                             |
| `J_VEC`                      | 3维实数向量 | `SUM_IVEC` × `A`                                             |
| `J_DOT_R`                    | 实数标量    | `SUM_R_DOT_IVEC` × `A`                                       |

### 算法步骤

**步骤 A：提取 RWG 参数**

- 若 `TRI_ID == RWG%POS_TRI_ID`：
  - `C` = `RWG%POS_COEF`
  - `R_OPP` = `POS_OPP_VERTEX` 坐标
- 否则：
  - `C` = `-RWG%NEG_COEF`（注意负号）
  - `R_OPP` = `NEG_OPP_VERTEX` 坐标

  **步骤 B：预计算常数**

- `JW_MU` = $(0.0, 1.0) \cdot K \cdot \text{ETA0}$
- `J_WE` = $(0.0, 1.0) \cdot \text{ETA0} / K$

**步骤 C：外层高斯积分（解析部分）**!

在三角形 $T$ 上取 $N$ 个高斯点（建议 **7点**）。

初始化 `SUM_I0` = 0, `SUM_R_I0` = (0,0,0), `SUM_IVEC` = (0,0,0), `SUM_R_DOT_IVEC` = 0。

对每个高斯点 $p$：
1. `R_VEC` = 第 $p$ 个高斯点全局坐标。
2. 调用 `ANALYTIC_SCALAR_POT_1_OVER_R` 得到 `I0_VAL`。
3. 调用 `ANALYTIC_LINEAR_POT_1_OVER_R` 得到 `I1_VAL`, `I2_VAL`, `I3_VAL`。
4. 计算向量势：`IVEC_VAL` = `V1`·`I1_VAL` + `V2`·`I2_VAL` + `V3`·`I3_VAL`。
5. 累加：
   - `SUM_I0` += `WEIGHTS(p)` × `I0_VAL`
   - `SUM_R_I0` += `WEIGHTS(p)` × `R_VEC` × `I0_VAL`
   - `SUM_IVEC` += `WEIGHTS(p)` × `IVEC_VAL`
   - `SUM_R_DOT_IVEC` += `WEIGHTS(p)` × `DOT_PRODUCT(R_VEC, IVEC_VAL)`

   循环结束后，统一乘以面积 `A`：
- `J0` = `SUM_I0` × `A`
- `J_R_I0` = `SUM_R_I0` × `A`
- `J_VEC` = `SUM_IVEC` × `A`
- `J_DOT_R` = `SUM_R_DOT_IVEC` × `A`

**步骤 D：Q算子 $1/R$ 解析部分**

$$Q_{\text{sing}} = -4 \cdot C^2 \cdot \frac{1}{4\pi} \cdot \text{J0}$$

即 `Q_SING` = `-4.0` × `C`² × `J0` / `(4.0 * PI)`

**步骤 E：P算子 $1/R$ 解析部分**

展开双线性项：
$$(\mathbf{r} - \mathbf{r}_{\text{opp}}) \cdot (\mathbf{r}' - \mathbf{r}_{\text{opp}}) = \mathbf{r} \cdot \mathbf{r}' - \mathbf{r} \cdot \mathbf{r}_{\text{opp}} - \mathbf{r}' \cdot \mathbf{r}_{\text{opp}} + |\mathbf{r}_{\text{opp}}|^2$$

利用累积量：
- `TERM1` = `J_DOT_R`（对应 $\int \mathbf{r} \cdot \mathbf{I}_{\text{vec}} dS$）
- `TERM2` = `- DOT_PRODUCT(R_OPP, J_R_I0)`（对应 $-\mathbf{r}_{\text{opp}} \cdot \int \mathbf{r} I_0 dS$）
- `TERM3` = `- DOT_PRODUCT(R_OPP, J_VEC)`（对应 $-\mathbf{r}_{\text{opp}} \cdot \int \mathbf{I}_{\text{vec}} dS$）
- `TERM4` = `DOT_PRODUCT(R_OPP, R_OPP) * J0`（对应 $|\mathbf{r}_{\text{opp}}|^2 \int I_0 dS$）

`P_SING` = `C`² × (`TERM1` + `TERM2` + `TERM3` + `TERM4`)

**步骤 F：$G_{\text{smooth2}}$ 数值部分**

对 $(T, T)$ 做双重高斯积分，核函数用 `GREEN_FUNC_SMOOTH2`。

复用 `CALC_GREEN_INTEGRALS` 框架，计算 $I_1, \mathbf{I}_2, \mathbf{I}_3, I_4$，然后按 EFIE 非对角线公式组合：
- `P_SMOOTH2` = $C^2 \cdot [|\mathbf{r}_{\text{opp}}|^2 I_1 - \mathbf{r}_{\text{opp}} \cdot \mathbf{I}_2 - \mathbf{r}_{\text{opp}} \cdot \mathbf{I}_3 + I_4]$
- `Q_SMOOTH2` = $-4 C^2 \cdot I_1$

**步骤 G：总阻抗**

```fortran
Z_PAIR = C * C * (JW_MU * (P_SING + P_SMOOTH2) + J_WE * (Q_SING + Q_SMOOTH2))
```

### 验证标准

- `P_SING` 通常为**正实数**。
- `Q_SING` 通常为**负实数**。
- `|P_SING|` > `|P_SMOOTH2|`，`|Q_SING|` > `|Q_SMOOTH2|`。
- `Z_PAIR` 为复数，实部为负。
- 模值远大于非对角线（5~20 倍）。

## 五、EFIE 相邻三角形自阻抗

### 目标

计算正三角形 $T^+$ 与负三角形 $T^-$ 之间的 EFIE 阻抗（共享一条边，弱奇异）。

### 接口

```fortran
SUBROUTINE CALC_EFIE_ADJACENT_TRI_PAIR(MESH, RWG, GDATA, K, ETA0, Z_PAIR)
```

### 算法步骤

1. 提取 `C_POS` = `RWG%POS_COEF`，`C_NEG` = `-RWG%NEG_COEF`（已含负号）。
2. 对 $(T^+, T^-)$ 调用 `CALC_GREEN_INTEGRALS`，使用**完整格林函数** `GREEN_FUNC`（不提取），高斯点用 **7点**。
3. 按 EFIE 非对角线公式组合得到 `Z_PM`。
4. 对 $(T^-, T^+)$ 同理得到 `Z_MP`。
5. `Z_PAIR` = `Z_PM` + `Z_MP`。

### 验证标准

- 相邻贡献模值小于同一三角形贡献。
- 正四面体中，结果应为复数。

## 六、MFIE 对角线元素

### 目标

计算 $Z^M_{mm}$。

### 接口

```fortran
SUBROUTINE CALC_MFIE_DIAGONAL_ELEMENT(MESH, RWG, GDATA, K, Z_MM)
```

### 变量定义

| 变量                     | 类型        | 含义             |
| :----------------------- | :---------- | :--------------- |
| `C_POS`, `C_NEG`         | 实数标量    | 正负三角形系数   |
| `A_POS`, `A_NEG`         | 实数标量    | 正负三角形面积   |
| `R_OPP_POS`, `R_OPP_NEG` | 3维实数向量 | 正负三角形对顶点 |
| `HALF_TERM`              | 实数标量    | 1/2 内积项累积   |
| `K_TERM`                 | 复数标量    | K算子贡献累积    |

### 算法步骤

**同一三角形对（正-正 + 负-负）**：
- K算子主值：**严格置 0**。
- 1/2 内积项：
  - 对 $T^+$：高斯积分 $\int_{T^+} |\mathbf{f}^{(+)}|^2 dS$，其中 $|\mathbf{f}^{(+)}|^2 = C_{\text{POS}}^2 \cdot |\mathbf{r} - \mathbf{r}_{\text{opp}}^{\text{POS}}|^2$。
  - 对 $T^-$：同理。
  - `HALF_TERM` = `0.5` × (正三角形积分 + 负三角形积分)。

  **相邻三角形对（正-负 + 负-正）**：
- K算子：复用现有 `CALC_MFIE_MATRIX_ELEMENT` 的 K算子逻辑（高斯积分）。
- 1/2 内积项：**严格为 0**（支撑域交集只有公共边）。

**总和**：
- `Z_MM` = `HALF_TERM` + `K_TERM`

### 验证标准

- `Z_MM` 为**正实数**。
- 模值约 $10^{-2} \sim 10^{-1}$（正四面体）。
- K算子贡献的模值远小于 1/2 项。

## 七、主程序集成

### EFIE 对角线组装

```fortran
DO M = 1, MESH%RWG_NUM
    RWG => MESH%RWG_BASES(M)
    
    CALL CALC_EFIE_SELF_TRI_PAIR(MESH, RWG%POS_TRI_ID, RWG, GDATA, K, ETA0, Z_PP)
    CALL CALC_EFIE_SELF_TRI_PAIR(MESH, RWG%NEG_TRI_ID, RWG, GDATA, K, ETA0, Z_MM)
    CALL CALC_EFIE_ADJACENT_TRI_PAIR(MESH, RWG, GDATA, K, ETA0, Z_PM)
    
    Z_E(M, M) = Z_PP + Z_MM + Z_PM
END DO
```

### MFIE 对角线组装

```fortran
DO M = 1, MESH%RWG_NUM
    CALL CALC_MFIE_DIAGONAL_ELEMENT(MESH, MESH%RWG_BASES(M), GDATA, K, Z_M(M,M))
END DO
```

### CFIE 组合

```fortran
ALPHA = 0.5
DO M = 1, MESH%RWG_NUM
    Z_CFIE(M, M) = ALPHA * Z_E(M, M) + (1.0 - ALPHA) * ETA0 * Z_M(M, M)
END DO
```

### 打印验证

```fortran
DO M = 1, MESH%RWG_NUM
    PRINT *, "RWG", M, &
             "EFIE对角:", ABS(Z_E(M,M)), &
             "MFIE对角:", ABS(Z_M(M,M)), &
             "CFIE对角:", ABS(Z_CFIE(M,M))
END DO
```
